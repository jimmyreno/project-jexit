# project-jexit

scaffolded using yeoman. uses browserify for local web-server.


currently deployed to https://jimmyreno.github.io

instructions

1) fork me

2) install packages
> npm install

3) get dependencies
> bower update

3) to run and display unit tests
> gulp serve:test

4) to display the code output
> gulp serve

5) to build a production version
> gulp dist
